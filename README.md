# data-structures and algorithms
Provide some implementations of standard data structures and algorithms written in javascript.

Data structures:
- heap (max)
- priority queue (max)
- stack
- queue
- linked-list
- binary tree

Algorithms (unit-tested):
- Comparision type:
	- insert sort
	- merge sort
	- bubble sort
	- quick sort

- Linear time type:
	- counting sort

- Search
	- binary search
